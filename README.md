# Kubernetes Monitoring Stack (Helm)

This project deploys a full Kubernetes monitoring & logging stack using Helm charts:

- **Prometheus** (kube-prometheus-stack)
- **Grafana**
- **Loki**
- **Promtail**
- **Alertmanager**

The stack provides:
- Metrics monitoring
- Log aggregation
- Dashboards (Grafana)
- Alerts (CPU, memory, latency)
- Node and Pod observability
